SpaceHey!

SpaceHey is een website online, maar ook app die in de categorie "Sociale Media" noemt. Het is een plaats waar je vrienden kan maken, creatief kan zijn, sturen met (nieuwe) vrienden, en op kan bloggen.

Je kan jouw profiel aanpassen, zodat het bij jouw eigen stijl en persoonlijkheid past. Hiervoor kan je ook een HTML code gebruiken om aangepaste elementen te bewerken en erin te steken. Je kan ook gewoon klikken op "Layouts" in de app/web en er ééntje kiezen om te kopiëren, als je niet goed bent int coderen.
